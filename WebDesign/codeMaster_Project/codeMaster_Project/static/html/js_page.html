Getting Started

Javascript is an easy-to-use programming language that can be embedded in the header of your web pages. It can enhance the dynamics and interactive features of your page by allowing you to perform calculations, check forms, write interactive games, add special effects, customize graphics selections, create security passwords and more.

With Javascript we will frequently to objects, events, methods, and properties, so first I will start by explaining what these keywords mean.

Objects/Properties
Your web page document is an object. Any table, form, button, image, or link on your page is also an object. Each object has certain properties (information about the object). This is the same concept as assigning CSS properties to your HTML objects.
Methods
Typical objects have a collection of things that they can do. Different objects can do different things, just like a light can turn on and off or a door can open and close. In javascript a new document is opened with the method document.open(). You can write "Hello World" into a document by typing document.write("Hello World"). both open() and write() are examples of methods of the object: document. Another word for methods is functions, so if you hear or read the word function, we are still talking about methods.
Events
Events are what triggers our functions. The easiest example of this is a button, whose definition includes the words onClick="doSomthingfunction()". The onClick event, as its name implies, will run the function when the user clicks on the button. Other common events include OnMouseOver, OnMouseOut, OnFocus, OnBlur, OnLoad, and OnUnload.

Javascript Syntax

Javascript syntax is much different than HTML and CSS and may take much longer to learn. Lets start with the basics; In your Javascript code you will often see the word "var". Var is short for variable, and a variable is basically a compartment that holds a piece of information. When we use javascript to create a program, and we want our program to remember information so that we can use it later, we assign it to a variable. For example the statement: var sayHello = "Hello World"; Stores the message hello world in memory. If later we want to print the message onto the screen we use the write function to do this: document.write(sayHello); There are many different types of variables in javascript but since this is an introductory site we'll only focus on two: strings, and numbers. In JS strings and numbers are assinged to variables almost the same, we start with "var" followed by the variable name. The variable name can be anything you want as long as it begins with a lowercase letter and does not contain spaces or special characters like exclamation points or question marks. The only special charachters allowed in varable names are dashes "-" and underscores"_", it is also good practice to make your variable names descriptive so that it is easy to tell what the variables are for. For example if I wanted to create a variable to hold a person's age I would declare my variable like this: var age=30; When declaring variables the difference between a string and a number is adding quotation marks for the string variable. for example- var name= "John"; var last-name="Smith"; var age="18"; The string value of the variable declaration is wrapped in quotation marks, while the number value is not.
