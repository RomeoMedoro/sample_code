What is HTML?

HTML is short for "HyperText Markup Language". It is the language used to describe what a web-page should look like using ordinary words. HTML is not a hard language, and it can be used to create websites.

HTML is a file document, every web-page is written using a text-editor (kind of like Microsoft Word). The file is what your browser uses to display the information on your screen.

What is CSS?

CSS is short for "Cascading Style Sheet". It is the language used to make your style as colorful and cool as you want.

Like HTML CSS is also a file document, and you can write it using a simple text-editor.

What is JAVASCRIPT?

Jacascript is what developers refer to a "scripting" language. Although you don't need to know Javascript to create websites, it is good to learn it.

Javascript is used to interact with the HTML and make your site much more fun than it otherwise would be. Javacript is also really important in creating games. Javascript is much more difficult to learn than HTML and CSS so it will require much more practice to master.
