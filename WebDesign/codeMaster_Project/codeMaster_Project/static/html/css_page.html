CASCADING STYLE SHEET

Cascading Style Sheets are what turn your website from a plain black and white site, to a beautiful and colorful site.

CSS just like HTML is a file, and just like with HTML you will be using a text editor to write your CSS code. CSS saves a lot of work because it can control the layout of multiple web pages all at once. The syntax (the way it is written) for CSS is much different than the syntax for HTML, but it is just as easy to write.

Instead of tags, in CSS we use the "selector" and curly brackets to apply the style. Below is an example of how we would apply a background color to the body of a site by using the body selector.

body{ background-color: green; }

The example above would apply a green background color to the entire body of the webpage. CSS can also be applied to elements using the "class" and "id" attributes of the element using the id or class attributes the css would look like this:.

.class-name{ background-color: green;}

#id-name{background-color: green;}

Notice the differences in the syntax for the three examples above, when applying css via tag selector you simply just name the element. This would apply the style you define in your curly brackets to everything inside the body of the HTML file. When applying css via class you use a period followed by the class name, when using the class to apply css, the styles will apply only to the elements with that class name. When applying css by id you use the hashtag # followed by the class name, just like with the class, using the id will only apply the style to the element with that specific id.

There are three ways that CSS can be applied to a site: externally, internally, and inline. Because externally is the preferred way especially for beginners, I will only show you how to use an external style sheet.

CSS syntax/

CSS can be used to make a wide range of changes to your site. CSS can tell the browser where to display elements, what background color to use, how much space should between each element, what font and font size to display and so on.

CSS is basically a list of declarations. Everything between the curly braces is called the declaration for that selector, and in the declaration there are a number of properties and values. There are several different properties and values available in CSS and learning them all will take time. W3Schools is a great place to find css properties and values. As you can see from the example you can apply properties to multiple selectors in a single declaration by simply separating each selector with a comma. Also pay attention to the way the colors and sizes are applied. Colors can be applied using english words such as color: red or background-color: red, or by using hex codes and rgb values. Hex codes are six character codes that start with a hashtag such as #009e0e, and rgb values determine to amount of red, green, and blue colors mixed together to make your final color. rgb values look like this: rgb(255, 50, 50). 255 is the max value of each color so by looking at the values in my example you can tell that the color would be mostly red because its values are 255 red, 50 green, and 50 blue. The higher all three values are, the closer to white your colors will get, and the lower the values, the closer the color gets to black. Black is written as rgb(0,0,0), and white is written as rgb(255, 255, 255).
