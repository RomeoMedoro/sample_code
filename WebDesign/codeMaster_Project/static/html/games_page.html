Lets practice writing code! In the container below you can see some html tags.

Lets start with something simple, inside the div element enter a "p" tag. In the tag write anything you want, make sure you close the tag when you are done. As you type your content take a look at your result window, you'll see your code automatically display.

Once you have completed this task, click next to go on to the next practice exercise

Inside the your new div create a button and give it an id and an onclick event called "myFunction()"

If you need some help click here hint

<button id="button" onclick="myFunction()" >. Make sure to add the closing tag, and be sure to add the button text between your opening and closing tag elements. This can be something as simple as "click me". When you are done it should look like this...

Great, lets add some styling to your button! Between the style tags enter your button's ID followed by opening and closing curly brackets. Give you button a background color of black, and text color white. Click here for a hint

Remember ids are preceded by a #, classes are preceded by a . elements are not preceded by anything...Another hint

background-color: black; color: white;

Now that you have a black button with white text, lets add some javascript to make the function work.

You'll notice in the editor we've already created script tags and, and we've started creating the function for you.

Inside the function's curly brackets type " alert('hi'); " then in the result window click the button

Great! Keep practicing html, css, and javascript on your own using our text editor!
