Html5 is not just used to create websites. The language is used to create a variety of games including games like Flappy Birds, Angry Birds, and Bejewled.

Learn at your own pace, and practice what you have learned in the "games and puzzles" portion of the site.

Getting Started

There are many programs that can be used to write computer language. These programs are often referred to IDEs which is short for Integrated Development Environment. IDEs are powerful programs that can be confusing for beginners to use. The cool thing with HTML is that all you need is a simple text editor.

Many conputers come with text editors already installed, the most common one is called Notepad. While Notepad is great for beginners, a better text editor would be Notepadd++. Notepad++ is free and easy to install and it is specifically designed for computer programming.

If you chose to use Notepad you are ready to move on to the next section; if you decide you'd prefer to use Notepad++ you can download it here: Download Notepad++

As you start using HTML to create your own sites, you can go to W3Schools for additional help, information, and resources.

HTML Basics

HTML tags are the hidden labels in a web page that tell your web browser how to display the content.

Most tags must have two parts, an opening and a closing part. For example, < html > is the opening tag and < /html > is the closing tag. Note that the closing tag has the same word as the opening tag, but has an additional forward-slash ( / ) character before the word. Think of the slash as the closing character.

There are some tags that do not require a closing tag, the < img > tag for showing images and the < br > tag for skipping to the next line are two examples. To create a webpage there are tags that you MUST include in your html file. These tags are known as essential tags and they are:

   < !doctype html >        Tells your browser this is an HTML5 file

   < html lang=”en” >      Tells your browser the language used in your site

   < head >                       Contains information about your site that will not display on the site itself

Although these tags are essential, only using these tags will simply create a blank screen, so the basic elements most commonly used to create a webpage that displays on the browser look like this:

In addition to the essential elements, an important tag to remember is the < div > tag. a div is more-or-less a container, and it is used often is creating a website. Also pay attention to the main, header, and footer tags, these tags are new tags and are part of the HTML5 syntax, so it is good to get into a habit of using them in your code.

Html elements can be assigned "class" and "id" attributes. These attributes will be important when you become more experienced, and will be used in applying CSS to the site, and for various functions using javascript. The "class" attribute can be given to an element, and that element can be used as often as you want throughout your page. The "id" attribute can be given to an element, but can only be used once for each page. Below are examples of using the class and id attributes.

< div class="class-name" > < /div >

< div id="id-name" > < /div >

My First Website

Follow these instructions to create your very first HTML page!!!

  • Open your text editor(Notepad, or Notepad++)
  • Enter the essential tags discussed in the previous section
  • Look at the exaple in the previus section, in the p tag where is says "main content goes here" enter Hello World!
  • Make sure you have entered everything correctly, use the code in the previous section as reference.
  • Click "file" and go to "Save As"
  • In the save file dialog select the location you want to save your file, in the File Name section, rename your file to MyFirstPage.html
    • The .html is the most important part. This is how the browser knows that it is a web-file.
  • Once the file is saved go to the location in which you've saved it and right-click it. Select "Open with" and click on a Web browser(edge, chrome, IE, etc.)

Once you've completed all the steps you'll see your first page open in your browser. Although there's not much to your first page, this is the way every website design begins.
